Bienvenido a SAETA API

Esta API permite acceder a datos de rutas de Villahermosa Tabasco, incluyendo sus detalles de ida, regreso y paradas.

Endpoints de la API

  • GET /api/route/:routeId/goings
    Obtiene todas las idas de una ruta específica.
  • GET /api/route/:routeId/going/:goingId
    Obtiene los detalles de una ida específica dentro de una ruta.
  • GET /api/route/:routeId/going/:goingId/stops
    Obtiene todas las paradas de una ida específica dentro de una ruta.
  • GET /api/route/:routeId/going/:goingId/stop/:stopId
    Obtiene los detalles de una parada específica dentro de una ida.
  • GET /api/route/:routeId/returns
    Obtiene todas las rutas de regreso de una ruta específica.
  • GET /api/route/:routeId/return/:returnId
    Obtiene los detalles de un regreso específico dentro de una ruta.
  • GET /api/route/:routeId/return/:returnId/stops
    Obtiene todas las paradas de un regreso específico dentro de una ruta.
  • GET /api/route/:routeId/return/:returnId/stop/:stopId
    Obtiene los detalles de una parada específica dentro de un regreso.

Reemplaza :routeId, :goingId, :returnId, y :stopId con los identificadores apropiados para realizar las consultas correspondientes.

Obtiene los detalles del regreso con ID 201 en la ruta con ID 1.

Solicitud

curl -X GET https://saeta-node-api.onrender.com/api/route/1/return/201
